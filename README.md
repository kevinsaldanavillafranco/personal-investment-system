# Personal Investment System (PIS)

A Python-based quantitative framework for portfolio risk analysis 
and investment strategy — built from the ground up as a structured 
learning project, with the goal of becoming a fully usable tool 
for systematic portfolio management.

## What this project is

PIS is a hands-on exploration of quantitative finance and data-driven 
investment analysis. It covers classical portfolio metrics — returns, 
volatility, correlation, Sharpe ratio, portfolio optimization — and 
progressively incorporates more data-driven methodologies from 
quantitative finance and applied data science.

The project follows a strict experimental methodology:

> hypothesis → experiment → validation → findings → interpretation

Every concept implemented is first formulated as a testable hypothesis, 
then verified computationally, and only then interpreted. Conclusions 
are earned, not assumed.

## What this project is not

This is not a trading system, a black-box strategy, or a finished product.
It is a rigorous, transparent, and evolving system — built to understand 
markets and portfolios from first principles.
